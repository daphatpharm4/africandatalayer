The system, briefly

A single-page React app with screen-based navigation — no router. App.tsx is the state hub. Vercel serverless functions back a Postgres (Supabase) primary store with Edge Config fallback.

Offline-first via IndexedDB queue. Bilingual EN/FR inline. Fraud detection, gamification, and RBAC across agent, admin, and client roles.

Use Diagram for layout, Nodes for the file inventory, Edges for relationships, Data & Services for storage and integrations. Spec: architecture.json.
