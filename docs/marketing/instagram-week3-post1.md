## Concept

Walk through one Bonamoussadi morning with an ADL field agent — 4 timestamps, 3 stops, real work, real rewards. Frame the agent role as a paid grassroots data craft, not a gig hustle stereotype. Recruitment payoff sits in slide 5; first 4 slides earn it by showing the work honestly.

**Why carousel:** Time-stamped sequence reads naturally as swipes. Each slide answers a recruit's silent question — "what does the day look like?", "what tech do I need?", "do I get paid for bad signal too?", "how fast does it land?", "what's in it for me?" Carousel re-serve to non-swipers also lifts reach versus a single-image post.

**Why now:** Week 3 = beta recruit window. LinkedIn handles the formal pitch the same week; IG carries the human, on-the-ground story. No avatar of a real agent — illustrated route + UI fragments only, per content rules.
