## Concept

Story sequence amplifies the launch carousel and converts taps into installs the same hour the feed post lands. Four frames: visual echo of the launch headline (frame 1), download link stack (frame 2), quiz that turns audience into participants (frame 3), DM-keyword question that opens a support thread for first-day captures (frame 4). Story format is doing the conversion work the carousel cannot — link sticker is the install path, quiz + question harvest commitment signals.

**Why story:** Launch-day install pressure happens in the first 6 hours. Stories sit at the top of the IG bar — they get tapped before the feed post is scrolled past. Quiz + link in one sequence = highest install velocity from a single post. Pinning to a fresh "LAUNCH" highlight extends the 24h ephemeral window into evergreen profile real estate for late-arriving traffic all week.
