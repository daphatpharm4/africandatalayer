# ADL Contributor Pitch Deck

Audience: Individual contributors (students, gig workers, local entrepreneurs)  
Goal: Drive trusted signups, first submission, and repeat weekly contribution.

## Slide 1
- Slide #: 1
- Title: Turn Local Knowledge Into Weekly Income
- Core message: ADL pays contributors to map real changes in their neighborhood.
- Slide copy: If you already know Bonamoussadi streets and businesses, you can convert that knowledge into weekly earnings. In Cameroon, many students run on monthly budgets of `30,000-80,000 FCFA`, and an extra `5,000-10,000 FCFA/month` can make a real difference [SM-29].
- Proof points:
  - Student budget and meaningful incremental earnings [SM-29].
  - Bonamoussadi pilot zone includes `~560` estimated registered businesses to map and update [SM-02].
- Suggested visual: Contributor story card with weekly activity and payout examples.
- Audience-specific CTA: "Join ADL and complete your first contribution this week."

## Slide 2
- Slide #: 2
- Title: How Contribution Works (Simple Workflow)
- Core message: Capture, submit, sync; ADL handles low-connectivity conditions.
- Slide copy: You capture a location, add details, and upload photo evidence. ADL is built offline-first and tuned for constrained data budgets: about `~9MB/day` typical usage and `~200MB/month` per active agent profile [SM-43].
- Proof points:
  - Typical data usage profile: `~9MB/day` [SM-43].
  - Monthly data profile per agent: `~200MB/month` [SM-43].
  - Sync failure operational target is `<2%` [SM-20].
- Suggested visual: Contributor flow diagram (Capture -> Queue -> Sync).
- Audience-specific CTA: "Try one text + photo submission and confirm it syncs."
