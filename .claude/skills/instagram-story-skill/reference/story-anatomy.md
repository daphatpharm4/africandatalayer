# Story Frame Archetypes

Every story frame fits one archetype. Mixing two archetypes in one frame splits attention and tanks tap-forward rate.

## Hook frame

**Purpose:** earn the second tap. Viewer decides in <1.5s whether to keep going.

- Background: high-contrast brand color (navy `#0f2b46` or terra `#c86b4a`).
- Headline: 56–80pt Inter 700, max 7 words.
- Visual anchor: one icon, one numeral, or one terra underline. Never two.
- Sticker: none. Friction kills the hook.
- Position: headline centered y=600–900.

## Proof frame

**Purpose:** earn trust before the ask. Use after hook, before CTA.

- Background: navy-wash `#f2f6fa` or forest-wash `#eaf1ec`.
- Headline: 44–56pt Inter 700, names the proof.
- Body: 22–26pt Inter 500, one sentence with the specific verification mechanism.
- Visual anchor: small icon chip (112×112) or accent bar (96×6).
- Sticker: optional location or hashtag at top of hero band.
- Position: stack from y=320 downward.

## Poll frame

**Purpose:** harvest engagement signal + audience preference data.

- Background: navy-wash or terra-wash.
- Headline: 36–48pt Inter 700, ends in a question.
- Sticker: poll at y=1100, 2 options max for stories (4 options for highlights).
- Body: skip — let the poll carry the frame.
- CTA: skip — the poll is the CTA.

## Quiz frame

**Purpose:** test audience knowledge, reward correct answers with affinity.

- Background: navy `#0f2b46` for high-stakes feel, navy-wash for casual.
- Headline: 36–44pt Inter 700, frames the question.
- Sticker: quiz at y=950, 3 options recommended.
- Set the correct answer in the IG composer — stick to one truthful answer per IG rules.

## Question frame

**Purpose:** open-ended audience harvest. Highest-effort engagement, lowest tap rate.

- Background: terra-wash `#fff8f4` or navy-wash.
- Headline: 36–44pt Inter 700, prompt is direct ("Which quartier should we map next?").
- Sticker: question at y=1100, prompt placeholder pre-filled in composer.
- Use sparingly — once per week max, since each question burns audience patience.

## Link frame

**Purpose:** drive to download / landing / external resource.

- Background: terra `#c86b4a` for highest tap rate (research-backed: warm colors lift link taps).
- Headline: 48–60pt Inter 700, names the destination ("Download today.").
- Visual anchor: arrow pointing down at the link sticker.
- Sticker: link at y=1500, custom text "Tap to download" (max 30 chars).
- Body: skip — link is the CTA.

## CTA frame

**Purpose:** non-link CTA — DM keyword, follow ask, share-to-story prompt.

- Background: terra `#c86b4a` or navy `#0f2b46`.
- Headline: 48–60pt Inter 700, clear ask.
- CTA pill: navy on terra OR gold on navy, y=1500.
- DM keyword: bold, set in quotes ("MAP" / "CARTE").
- Sticker: optional mention sticker for handle reinforcement.

## Recommended sequences

| Story length | Sequence |
|--------------|----------|
| 3 frames | Hook → Proof → CTA |
| 4 frames | Hook → Proof → Poll → CTA |
| 5 frames | Hook → Proof → Proof → Poll/Question → Link |
| 6 frames | Hook → Proof → Proof → Quiz → Question → Link |
| 7+ frames | Use carousel skill instead — story attention dies past 6. |

## Hold time per archetype (for posting plan)

| Archetype | Default hold |
|-----------|--------------|
| Hook | 5s (IG default) |
| Proof | 7s (denser text) |
| Poll/Quiz | 15s (interactive) |
| Question | 15s (interactive) |
| Link | 5s |
| CTA | 5s |

Adjust hold via IG composer `Duration` setting — default 5s does not give text-heavy proof frames enough time.
